FIPE HTTP REST

Você pode usar a biblioteca em PHP desenvolvida para consumir a API https://github.com/deividfortuna/fipe.

A API disponibiliza seus dados de busca no formato JSON. Confira a URL base de acesso a API:

https://fipe-parallelum.rhcloud.com/api/v1

Marcas

Primeiro liste as marcas do tipo de veículo que você deseja, através da ação marcas e sem nenhum parâmetro:

Request

GET: https://fipe-parallelum.rhcloud.com/api/v1/carros/marcas

Você pode alterar na url "carros" para "motos" ou "caminhoes" de acordo com a sua necessidade.

Reponse

[ { "nome": "Acura", "codigo": 1 }, { "nome": "Agrale", "codigo": 2 }, { "nome": "Alfa Romeo", "codigo": 3 }, { "nome": "AM Gen", "codigo": 4 }, ... ]

Modelos

Na sequencia você poderá obter a listagem de veículos de uma determinada marca, com o código da marca desejada. Por exemplo a marca VW - VolksWagen (59):

Request

GET: https://fipe-parallelum.rhcloud.com/api/v1/carros/marcas/59/modelos

Reponse

{ "modelos": [ { "nome": "AMAROK CD2.0 16V/S CD2.0 16V TDI 4x2 Die", "codigo": 5585 }, { "nome": "AMAROK CD2.0 16V/S CD2.0 16V TDI 4x4 Die", "codigo": 5586 }, { "nome": "AMAROK CS2.0 16V/S2.0 16V TDI 4x2 Diesel", "codigo": 5748 }... ], "anos": [ { "nome": "32000 Gasolina", "codigo": "32000-1" }, { "nome": "32000 Diesel", "codigo": "32000-3" }, { "nome": "2016 Gasolina", "codigo": "2016-1" }, { "nome": "2016 Diesel", "codigo": "2016-3" }... ] }

Anos

Após escolher o veículo desejado é possível consultar os modelos e os anos disponíveis para uma futura consulta de preços. Por exemplo AMAROK High.CD 2.0 16V TDI 4x4 Dies. Aut:

Request

GET: https://fipe-parallelum.rhcloud.com/api/v1/carros/marcas/59/modelos/5940/anos

Reponse

[ { "nome": "32000 Diesel", "codigo": "32000-3" }, { "nome": "2016 Diesel", "codigo": "2016-3" }, { "nome": "2015 Diesel", "codigo": "2015-3" }... ]

Valor

Por fim adicionando mais um parâmetro será possível visualizar o preço corrente da Tabela FIPE para este veículo/modelo/ano. Continuando com o exemplo a cima para obter o valor de um veículo do ano 2014 a Gasolina utilizaremos o id 2014-3:

Request

GET: https://fipe-parallelum.rhcloud.com/api/v1/carros/marcas/59/modelos/5940/anos/2014-3

Reponse

{ "Valor": "R$ 119.019,00", "Marca": "VW - VolksWagen", "Modelo": "AMAROK High.CD 2.0 16V TDI 4x4 Dies. Aut", "AnoModelo": 2014, "Combustivel": "Diesel", "CodigoFipe": "005340-6 ", "MesReferencia": "November de 2015 ", "TipoVeiculo": 1, "SiglaCombustivel": "D" }

Com isto descubrimos os detalhes do veículo AMAROK High.CD 2.0 16V TDI 4x4 Dies. Aut ano 2014 a Diesel.
